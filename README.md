学习编程我们要建立四个好习惯

1.一个习惯就是下载软件要去这个软件的官网下载，这样你永远可以下载到最新的 最正宗的

2.阅读英文的网站 阅读英文的文献 使用英文的软件

3.写程序的人一定要习惯使用键盘做事情，而不要时时用鼠标。因为你只有两只手，没有第三支手，所以程序员一般都习惯所有的事情都在键盘上做。

4 学计算机要放心大胆的去尝试。去想 去思考
